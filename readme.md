# RNN Implementation

This project demonstrates the implementation of a **Recurrent Neural Network (RNN)** for basic sequence processing and learning tasks using Python.

## 📌 Project Overview

Recurrent Neural Networks are widely used for handling **sequential data** such as text, time series, and speech.
This notebook shows how an RNN works internally and how it can be implemented step-by-step.

## ⚙️ Technologies Used

* Python
* NumPy
* Jupyter Notebook

## 🚀 How to Run

1. Clone the repository

```
git clone https://github.com/shivamparihari-46/RNN_implementation.git

```

3. Open the notebook

```
jupyter notebook
```

4. Run **RNNimplementation.ipynb**
